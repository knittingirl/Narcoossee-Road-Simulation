# Narcoossee-Road-Simulation
This is for the COT6571 course project. 

## Running

Several versions of the simulation testing various parameter changes are available, currently labelled runner_lines_*.py, with a reasonably descriptive name for the tests that the file makes. Simply run the appropriate python file to duplicate research results. Note that cached versions of the FDOT data are provided; if fresh data pulled directly from the FDOT website is desired, remove both .json files.

## Results

The full results of our simulation runs are available in the outputs directory. This includes csv files with full data from the runs as well as line graphs plotting some of the most salient results.
